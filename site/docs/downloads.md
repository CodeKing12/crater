## System Requirements

### Minimum Requirements

- **OS:** Windows 10, macOS 10.13+, or Linux (Ubuntu 18.04+)
- **RAM:** 4 GB
- **Storage:** 500 MB available space
- **Display:** 1280 x 720 resolution

### Recommended Requirements

- **RAM:** 8 GB or more
- **Display:** Dual monitor setup (one for controls, one for projection)

---

## Installation Instructions

### Windows

1. Download the `.exe` installer
2. Run the installer and follow the prompts
3. Launch Crater from the Start Menu or Desktop shortcut

### macOS

1. Download the `.dmg` file
2. Open the disk image
3. Drag Crater to your Applications folder
4. Launch from Applications (you may need to right-click and select "Open" the first time)

### Linux

**AppImage:**

1. Download the `.AppImage` file
2. Make it executable: `chmod +x Crater*.AppImage`
3. Run the AppImage: `./Crater*.AppImage`

**Debian/Ubuntu:**

1. Download the `.deb` file
2. Install via terminal: `sudo dpkg -i Crater*.deb`
3. Launch from your applications menu
