# Application Settings

This guide covers all the settings available in Crater and how to configure them for your setup.

## Accessing Settings

1. Click the **Settings** icon (gear) in the menu bar
2. The Settings dialog opens
3. Configure your options
4. Close the dialog to save

## Display Settings

### Select Display

Choose which monitor/projector to use for the projection output.

**How to configure:**

1. Make sure your projector or second monitor is connected
2. Open Settings
3. Find "Select Display"
4. Click the dropdown
5. Choose your projection display from the list

**Tips:**
- If your display isn't listed, check the cable connection
- Restart Crater if you connected a display after opening the app
- The primary display is usually your laptop/computer screen

### Display Bounds

These fields show the dimensions and position of the selected display:

| Field | Description |
|-------|-------------|
| Left | Horizontal position from the left edge |
| Top | Vertical position from the top edge |
| Width | Display width in pixels |
| Height | Display height in pixels |

These are automatically set when you select a display and typically don't need manual adjustment.

## Projection Window

### Opening the Projection Window

1. Click the projection icon in the menu bar
2. A new window opens on your selected display
3. This window shows your live content to the audience

### Closing the Projection Window

1. Click the projection icon again
2. Or close the projection window directly

**Note:** The projection window should remain open during your service.

## Theme Settings

### Default Scripture Theme

The theme applied when displaying Bible verses.

**To change:**
1. Go to the Themes tab
2. Right-click on a Scripture theme
3. Select "Set as Scripture Theme"

### Default Song Theme

The theme applied when displaying song lyrics.

**To change:**
1. Go to the Themes tab
2. Right-click on a Song theme
3. Select "Set as Song Theme"

### Default Presentation Theme

The theme applied for general presentations.

**To change:**
1. Go to the Themes tab
2. Right-click on a Presentation theme
3. Select "Set as Presentation Theme"

## Logo Settings

### What is the Logo?

The logo is a background image shown when:
- The Clear/Logo button is pressed
- No content is being displayed
- You want to show a default background

### Setting a Logo Image

1. Go to the Media tab
2. Find the image you want as your logo
3. Right-click on it
4. Select "Set as Logo"

### Recommended Logo Specifications

| Property | Recommendation |
|----------|----------------|
| Resolution | Match your projector (e.g., 1920x1080) |
| Aspect Ratio | 16:9 for widescreen projectors |
| Format | JPEG or PNG |
| Content | Your church logo, subtle background |

## Display Configuration Guide

### Before First Use

1. Connect your projector/second monitor
2. Set Windows to "Extend" display mode
3. Open Crater Settings
4. Select the correct display
5. Open the projection window
6. Test with sample content

### Checking Your Setup

1. The projection window should appear on your projector
2. Content should display correctly
3. Text should be readable from the back of the room

## Common Settings Tasks

### Change the Projection Display

When you need to use a different projector:

1. Connect the new display
2. Wait for Windows to detect it
3. Open Crater Settings
4. Select the new display from the dropdown
5. Close and reopen the projection window

### Reset to Default Settings

If settings become corrupted:

1. Close Crater completely
2. Locate the settings file in your app data folder
3. Delete or rename the settings file
4. Restart Crater (defaults will be restored)

### Troubleshoot Display Issues

If the projection isn't working:

1. Check cable connections
2. Verify Windows detects the display
3. Open Settings and select the correct display
4. Close and reopen the projection window
5. Restart Crater if needed

## Settings Best Practices

### Regular Checks

- Verify display settings before each service
- Test the projection window early
- Have a backup plan if technology fails

### Environment Considerations

- Room lighting affects readability
- Projector brightness may need adjustment
- Test at the same time as your actual service
