# Keyboard Shortcuts

Master these keyboard shortcuts to operate Crater quickly and efficiently during services.

## Navigation Shortcuts

### Arrow Keys

| Key | Action |
|-----|--------|
| ↓ Arrow Down | Move to next item / Next verse or section |
| ↑ Arrow Up | Move to previous item / Previous verse or section |
| ← Arrow Left | (Context dependent) |
| → Arrow Right | (Context dependent) |

### Selection

| Key | Action |
|-----|--------|
| Enter | Confirm selection / Go live with current item |

## Panel Navigation

### Understanding Focus

Crater uses a "focus" system. The panel with focus receives your keyboard input.

- Click on a panel to give it focus
- The focused panel has a visual indicator
- Keyboard commands affect the focused panel

### Focus Flow

1. **Content Tabs** - Where you browse songs, scripture, etc.
2. **Schedule Panel** - Your service order list
3. **Preview Panel** - Content preview area
4. **Live Panel** - Currently displayed content

## Quick Reference Card

### During Service

| Goal | Action |
|------|--------|
| Preview content | Single-click on item |
| Display content | Double-click on item |
| Next verse/section | ↓ Arrow Down |
| Previous verse/section | ↑ Arrow Up |
| Confirm/Go live | Enter |

### Display Controls

| Goal | Action |
|------|--------|
| Clear display | Click Clear button |
| Show logo | Click Logo button |
| Open projection | Click projection button |

## Working in Different Panels

### In the Songs Tab

| Key | Action |
|-----|--------|
| ↓ | Move down in song list |
| ↑ | Move up in song list |
| Enter | Select highlighted song |

### In the Scripture Tab

| Key | Action |
|-----|--------|
| ↓ | Move down in verse list |
| ↑ | Move up in verse list |
| Enter | Select highlighted verse |

### In the Schedule Panel

| Key | Action |
|-----|--------|
| ↓ | Move down in schedule |
| ↑ | Move up in schedule |
| Enter | Preview/select item |

### In the Preview Panel

| Key | Action |
|-----|--------|
| ↓ | Highlight next verse/section |
| ↑ | Highlight previous verse/section |
| Enter | Send to live display |

### In the Live Panel

| Key | Action |
|-----|--------|
| ↓ | Display next verse/section |
| ↑ | Display previous verse/section |

## Tips for Operators

### Practice These Key Combinations

1. **Arrow Down** → Next content
2. **Arrow Up** → Previous content
3. **Enter** → Confirm selection

These three actions handle 90% of your service operation.

### Muscle Memory

- Practice navigating before services
- Get comfortable with the flow
- Speed comes with repetition

### During Service Best Practices

1. Keep one hand on the arrow keys
2. Watch the worship leader for cues
3. Stay one step ahead in your mind
4. Don't rush - smooth transitions are better than fast ones

## Mouse + Keyboard Combination

The most efficient operators use both:

| Task | Best Method |
|------|-------------|
| Browse content | Mouse scrolling |
| Select items | Mouse click |
| Navigate verses | Arrow keys |
| Quick adjustments | Arrow keys |
